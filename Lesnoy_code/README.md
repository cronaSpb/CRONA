## Настройка сервера

### 1. Загрузка файлов
Загрузите всё содержимое папки `Lesnoy_code` в корень сайта `bio-plus.ru`

### 2. Права доступа
Установите права `755` на папки и `644` на файлы

### 3. PHP настройки
Убедитесь что на сервере:
- PHP 7.4+ установлен
- Расширение `curl` включено (для API)
- Расширение `json` включено

### 4. База данных (опционально)
Если нужна база данных для заявок:
```sql
CREATE TABLE requests (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(255),
    phone VARCHAR(20),
    email VARCHAR(255),
    message TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## Проверка работоспособности

### 1. Модальные окна
Откройте `H2O/H2O.html` и проверьте:
- Кнопки "Подробнее" открывают модальные окна
- Модальные окна закрываются по клику на × или вне окна
- Анимации не просвечивают через модальные окна

### 2. AI ассистент
Проверьте работу AI ассистента:
- Открывается по клику на иконку
- Отправляет запросы к API
- Отображает ответы

### 3. Формы обратной связи
Проверьте отправку форм:
- Заполните форму на главной странице
- Проверьте получение письма на почту
